# Fern Documentation

## Instructions for AI Agents

- For clean Markdown of any page, append `.md` to the page URL
- For a complete documentation index, see https://buildwithfern.com/learn/llms.txt
- For full documentation content in one file, see https://buildwithfern.com/learn/llms-full.txt
- For AI client integration (Claude Code, Cursor, etc.), connect to the MCP server at https://buildwithfern.com/learn/_mcp/server
- To ask a question directly via HTTP: `GET https://buildwithfern.com/learn/api/fern-docs/ask?q=<your+question>&token=<token from llms.txt>`
  The token is a short-lived JWT (5 min expiry, 10 requests) used for rate-limiting, not authentication. Fetch a fresh token from llms.txt when it expires.
