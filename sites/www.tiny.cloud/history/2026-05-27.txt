# TinyMCE – The Most Trusted & Feature-rich WYSIWYG Rich Text Editor


> TinyMCE is an advanced WYSIWYG Rich Text Editor trusted by over 1.5M+ developers worldwide, with a platform that combines AI-powered tools, seamless integrations, and extensive customization for a superior rich text editing experience. TinyMCE offers broad flexibility, making it a powerful solution that adapts to a wide range of industries and roles. Its modular design, extensive plugin ecosystem, and seamless integration capabilities allow you to tailor the editor to meet diverse needs—from simple websites to complex enterprise applications—emphasizing its adaptability and design freedom.


## Core Benefits
- Seamless Integration: Easily embed TinyMCE into any web or mobile application with support for Angular, React, Vue, jQuery, Node.js, and more.
- AI-Powered Editing: TinyMCE AI adds document-aware chat, customizable quick actions, and an inline review mode for grammar, style, tone, and readability — fully managed with no API keys required.
- Highly Customizable: Tailor the editor’s functionality and appearance to your exact needs with a vast ecosystem of plugins, themes, and APIs.
- Enterprise-Grade Security: Built to meet high security and compliance standards, ensuring your data and user content remain safe.
- Rapid Deployment: Benefit from out-of-the-box features and comprehensive documentation that get you up and running quickly.


## Key Features
- Intuitive WYSIWYG Interface: Real-time rich text editing with familiar, user-friendly controls.
- Responsive & Adaptive Design: Optimized for performance across desktops, tablets, and smartphones.
- Advanced Formatting & Media Management: Robust text formatting, tables, lists, and integrated file & image management via Moxie Manager.
- Custom Plugins & API Access: Extend functionality and seamlessly integrate with your back-end systems through our extensive API and plugin support.
- Collaboration & Workflow Tools: Enhance team productivity with built-in commenting, version history, and workflow integrations.


## Tailored Solutions
- Enterprise Solutions: Scalable and secure deployments for large organizations with custom requirements.

- Content Creators & Marketers: Rich, intuitive tools designed to streamline digital content creation and management.

## Core Editing Features
- Basic Font Formatting: Enables basic font formatting options such as bold, italics, and underline.
- [Insert Media](https://www.tiny.cloud/docs/tinymce/latest/media/): Provides users with the ability to add HTML5 video and audio elements to the editable area.
- [Insert Link](https://www.tiny.cloud/docs/tinymce/latest/link/): Allows a user to link external resources such as website URLs, to selected text in their document.
- [Insert Date/Time](https://www.tiny.cloud/docs/tinymce/latest/insertdatetime/): Provides a toolbar control and menu item that lets a user easily insert the current date and/or time into the editable area at the cursor insertion point.
- [Code Sample](https://www.tiny.cloud/docs/tinymce/latest/codesample/): Lets a user insert and embed syntax color highlighted code snippets into the editable area.
- [Emoticons](https://www.tiny.cloud/docs/tinymce/latest/emoticons/): Adds a dialog to the editor that lets users insert emoji into TinyMCE’s editable area.
- [Character Map](https://www.tiny.cloud/docs/tinymce/latest/charmap/): Adds a dialog to the editor with a map of special unicode characters, which cannot be added directly from the keyboard.
- [Visual blocks](https://www.tiny.cloud/docs/tinymce/latest/visualblocks/): Allows a user to see block level elements in the editable area. It also adds a toolbar button and a menu item Show blocks under the View menu dropdown.
- [Visual characters](https://www.tiny.cloud/docs/tinymce/latest/visualchars/): Adds the ability to see invisible characters like “&nbsp;” displayed in the editable area. It also adds a toolbar control and a menu item Show invisible characters under the View menu.
- [Directionality](https://www.tiny.cloud/docs/tinymce/latest/directionality/): Adds directionality controls to the toolbar, enabling TinyMCE to better handle languages written from right to left. It also adds a toolbar button for each of its values, ltr for left-to-right text and rtl for right-to-left text.
- [Import CSS](https://www.tiny.cloud/docs/tinymce/latest/importcss/): Adds the ability to automatically import CSS classes from the CSS file specified in the content_css configuration setting.
- [Basic Code Editor](https://www.tiny.cloud/docs/tinymce/latest/code/): Adds a toolbar button that allows a user to edit the HTML code hidden by the WYSIWYG view.
- [Help](https://www.tiny.cloud/docs/tinymce/latest/help/): Adds a button and/or menu item that opens a dialog showing four tabs: Handy shortcuts, Keyboard navigation, Plugins, Version.
- [Text Pattern](https://www.tiny.cloud/docs/tinymce/latest/content-behavior-options/#text_patterns): allows configuring the text patterns that get matched while typing in the editor. Text patterns can be used to apply formats, replace text, or execute commands.
- [Text Color](https://www.tiny.cloud/docs/tinymce/latest/user-formatting-options/#text-color-options): affect the color selector shown when using the forecolor (text color) and backcolor (text background) toolbar buttons or menu items. The dimensions and mapping of the grid of text colors can be set here.
- [Color Picker](https://www.tiny.cloud/docs/tinymce/latest/dialog-components/#colorpicker): an intuitive color picker tool similar to that found in modern image editors. It allows for a color to be chosen using a RGB colour slider, or for a color to be entered as either an RGB or hex color value.
- [Insert Image](https://www.tiny.cloud/docs/tinymce/latest/image/): Enables a user to insert an image into TinyMCE’s editable area.
- [Non-editable](https://www.tiny.cloud/docs/tinymce/latest/non-editable-content-options/): allows you to specify the class name that TinyMCE will use to determine which areas of content are editable.
- [Save](https://www.tiny.cloud/docs/tinymce/latest/save/): Adds a save button to the TinyMCE toolbar, which will submit the form that the editor is within.

- [Context Menu](https://www.tiny.cloud/docs/tinymce/latest/contextmenu/): a configurable component that appears when the user right clicks in the editable area. By default it does not disable the operating system’s native context menu, if there are no items to display at the cursor position the native context menu will be shown.
The context menu supports both individual menu items and dynamic context menu sections.
- [Quick Toolbars](https://www.tiny.cloud/docs/tinymce/latest/quickbars/): The Quick Toolbar plugin adds three context toolbars: a Quick Selection toolbar - shown when text is selected, providing formatting buttons such as: bold, italic, and link; a Quick Insert toolbar - shown when a new line is added, providing buttons for inserting objects such as tables and images; a quick Image toolbar - Shown when an image or figure is selected, providing image formatting buttons such as alignment options. The feature also adds three toolbar buttons: Quick Link - An inline form for creating and editing links without a dialog; Quick Image - Prompts the user to select a local image to upload; Quick Table - Inserts a 2x2 table without prompting the user to select the number of rows and columns.
- [Autolink](https://www.tiny.cloud/docs/tinymce/latest/autolink/): Automatically creates hyperlinks when a user types a valid, complete URL.
- [Enhanced Tables](https://www.tiny.cloud/tinymce/features/enhanced-tables/): Built to look, and feel, like the most popular spreadsheet tools, this feature combines familiarity with more advanced functionality.
- [Merge Tags](https://www.tiny.cloud/tinymce/features/merge-tags/): Insert predefined, read-only merge tags (also known as personalization tokens, or mail merge fields) into your content
- [Enhanced Code Editor](https://www.tiny.cloud/tinymce/features/enhanced-code-editor/): An IDE-like HTML editor right within your editor
- [Image Editing](https://www.tiny.cloud/tinymce/features/image-editing/): Users can resize, rotate, crop, flip, invert or adjust the brightness, contrast, color levels, gamma, or sharpness of their images.
- [Math Equations](https://www.tiny.cloud/tinymce/features/math-equations/): Add formulas and equations to your content using LaTex or MathML.
- [Case Change](https://www.tiny.cloud/tinymce/features/case-change/): This feature lets you toggle selected text between Title Case, UPPERCASE, and lowercase and is accessible via a toolbar button or the Format menu.
- [Format Painter](https://www.tiny.cloud/tinymce/features/format-painter/): Copy-paste formatting, such as font style and size, without reformatting
- [Permanent Pen](https://www.tiny.cloud/tinymce/features/permanent-pen/): With the feature toggled on, the style that’s set is automatically applied to all text entered in the document. Being able to store reusable information saves you time and ensures consistency across the content.
- [PowerPaste](https://www.tiny.cloud/tinymce/features/powerpaste/): With 99.9% accuracy, it’s the only rich text editor feature on the market that lets users copy-paste clean content, without developers needing to worry about breaking the underlying HTML code.
- [Templates](https://www.tiny.cloud/tinymce/features/templates/): Create, organize, and insert content snippets in seconds
- [Media Optimizer powered by Uploadcare](https://www.tiny.cloud/tinymce/features/media-optimizer/): Simplifies the uploading, editing, and optimization of both images and videos—delivering sharper visuals, smoother playback, and faster load times. 

## TinyMCE AI
- [TinyMCE AI](https://www.tiny.cloud/tinymce/features/ai-content-editor/): A fully managed AI plugin with persistent, document-aware chat, customizable quick actions for rewriting and tone adjustment, and a review mode that surfaces inline suggestions for grammar, style, tone, and readability. Supports OpenAI, Anthropic Claude, and Google Gemini — no API keys required.
- [TinyMCE AI Docs](https://www.tiny.cloud/docs/tinymce/latest/tinymceai-introduction/): Overview of TinyMCE's native AI capabilities — bring your own model (BYOM), contextual content generation, editing, and review, embedded directly in the editor.
- [Chat](https://www.tiny.cloud/docs/tinymce/latest/tinymceai-chat/): Conversational AI interface inside the editor. Users generate, refine, and insert content through multi-turn prompts without leaving the writing surface.
- [Review](https://www.tiny.cloud/docs/tinymce/latest/tinymceai-review/): AI-powered content review that suggests improvements to grammar, clarity, tone, and structure with inline accept/reject controls.
- [Actions](https://www.tiny.cloud/docs/tinymce/latest/tinymceai-actions/): Pre-configured and custom AI commands (summarize, translate, expand, change tone) exposed as one-click toolbar or menu items.
- [AI Models](https://www.tiny.cloud/docs/tinymce/latest/tinymceai-models/): Connect TinyMCE AI to any supported LLM provider — OpenAI, Azure OpenAI, Amazon Bedrock, Google Gemini — or route through a custom proxy.
- [Limits](https://www.tiny.cloud/docs/tinymce/latest/tinymceai-limits/): Usage, token, and rate-limit guidance for production deployments.
- [Integration Options](https://www.tiny.cloud/docs/tinymce/latest/tinymceai-integration-options/): Deployment patterns — direct provider integration, server-side proxy, and enterprise architectures for data governance.
- [JWT Authentication](https://www.tiny.cloud/docs/tinymce/latest/tinymceai-jwt-authentication-intro/): Secure the AI request pipeline with JWT-based authentication between the editor, your backend, and the model provider.
- [OpenAI (ChatGPT) Integration](https://www.tiny.cloud/docs/tinymce/latest/ai-openai/): Step-by-step guide to connecting TinyMCE AI with OpenAI / ChatGPT models.
- [Azure OpenAI Integration](https://www.tiny.cloud/docs/tinymce/latest/ai-azure/): Connect TinyMCE AI to Azure OpenAI deployments for enterprise-grade compliance and regional data residency.
- [Amazon Bedrock Integration](https://www.tiny.cloud/docs/tinymce/latest/ai-bedrock/): Route AI requests through Amazon Bedrock to access Anthropic Claude, Meta Llama, and other foundation models.
- [Google Gemini Integration](https://www.tiny.cloud/docs/tinymce/latest/ai-gemini/): Use Google Gemini models as the LLM backend for TinyMCE AI features.
- [AI Proxy Server Reference](https://www.tiny.cloud/docs/tinymce/latest/ai-proxy/): Reference implementation for a server-side proxy that brokers AI requests, handles authentication, and keeps provider credentials off the client.
- [AI Assistant (Legacy)](https://www.tiny.cloud/docs/tinymce/latest/ai/): Legacy AI Assistant plugin — the original dialog-based integration. Retained for backward compatibility; new projects should use the modern TinyMCE AI stack above.
