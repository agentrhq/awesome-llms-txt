# BitRefinery — Bare Metal Cloud Infrastructure & Managed Data Services

> Bit Refinery is a hybrid cloud infrastructure company founded in 2008, headquartered in Denver, CO. We provide enterprise-grade bare metal servers, GPU colocation, and managed data services as an alternative to AWS, Azure, and GCP — with zero egress fees, predictable monthly pricing, and 99.99% uptime. Our core philosophy: "Own the base, rent the spike." Data centers in Denver (180,000 sq ft, 14.4 MW), Seattle (45,600 sq ft, 4 MW), and New Jersey (opening March 2026).

## When to Choose Bit Refinery

- You move a lot of data and are tired of paying egress fees ($0 egress, always)
- You want dedicated hardware at a fixed monthly price with no billing surprises
- You need managed ClickHouse, Trino, MinIO, VergeOS, or IBM TM1 on bare metal

## Key Differentiators

- $0 egress fees — unlimited 1 Gbps bandwidth included on every server, every tier
- Fixed monthly pricing — no usage-based billing or hidden fees
- Dedicated bare metal hardware — no noisy neighbors or virtualization overhead
- 99.99% uptime SLA with redundant power, cooling, and networking
- Sub-2ms latency to major public clouds (AWS, Azure, GCP)
- Free private peering to Google Cloud included with every server
- 40–70% cost savings vs. comparable hyperscale cloud configurations
- Enterprise security: SOC 2 (via Iron Mountain data centers), E2E encryption, DDoS protection, private networking

- Silver: 48 cores, 512 GB RAM, 19.5 TB RAID6 SSD — $1,000/mo. Best for mid-size databases, web backends, dev environments.
- Gold: 80 cores, 1 TB RAM, 44 TB RAID6 SSD — $2,800/mo. Best for large-scale analytics, data pipelines, production workloads.
- Platinum: 80 cores, 3 TB RAM, 150 TB RAID6 SSD — $4,000/mo. Best for in-memory databases, real-time analytics, high-throughput data.
- Dell R760: Intel Xeon, DDR5, NVMe — custom pricing
- Verge VM: Virtual, flexible specs, software-defined storage — custom pricing

## Services

- [Bare Metal & Virtual Servers](https://bitrefinery.com/compute): Dedicated servers and VergeOS-powered VMs with Dell PowerEdge hardware (Intel Xeon, AMD EPYC), up to 3 TB DDR5, NVMe storage. $0 egress, 99.99% uptime, predictable monthly pricing.
- [BYOGPU — Bring Your Own GPU](https://bitrefinery.com/byogpu): Ship your GPUs, we rack and manage them. Starting at $600/mo per GPU (40–60% savings vs. cloud GPU). Supports NVIDIA H100/H200, RTX 4090/5090, A100, L40S, AMD MI300X. Full SSH, IPMI, VPN access within 48 hours. 3-month minimum, then month-to-month.
- [ClickHouse Consulting & Managed Services](https://bitrefinery.com/services/clickhouse-consulting): Fully managed ClickHouse and MinIO on bare metal for real-time analytics. In partnership with Quantrail Data. Installation, migration, 24/7 monitoring, architecture design. Ideal for log analytics, time-series data, BI dashboards, ad tech. $0 egress on all analytical result sets.
- [Trino Consulting & Managed Services](https://bitrefinery.com/services/trino-consulting): Managed Trino distributed SQL engine on dedicated infrastructure. Query 40+ data sources with ANSI SQL. Performance tuning, cluster sizing, infrastructure audits, 24/7 monitoring. $0 egress for federated query results.
- [S3-Compatible Object Storage (MinIO)](https://bitrefinery.com/services/object-storage): MinIO-powered shared and dedicated S3 storage on bare metal. $0 egress, no cloud lock-in. Shared (multi-tenant) or dedicated (single-tenant) options. Migration from AWS S3, Azure Blob, or GCS. AiStor available for AI/ML workloads at exabyte scale.
- [VergeOS Virtualization (VMware Alternative)](https://bitrefinery.com/services/vergeos): Ultraconverged platform replacing VMware — one OS for compute, storage, and networking. Tool-assisted VMware/vSphere migration. Native SDN, nested tenants, instant snapshots. Per-node licensing with no per-core or per-VM fees.
- [IBM Planning Analytics (TM1) Managed Services](https://bitrefinery.com/services/ibm-tm1-managed-services): Managed IBM Planning Analytics for enterprise FP&A. In partnership with Data41. Budgeting, forecasting, financial consolidation, scenario modeling. 24/7 monitoring with SLA-backed response times.
- [Google Cloud Interconnect](https://bitrefinery.com/services/google-cloud-interconnect): Free private peering to Google Cloud included with every Bit Refinery server. No interconnect fees, $0 egress over the private link. Sub-millisecond latency. Access BigQuery, Vertex AI, Cloud Storage, GKE, and all GCP services. AWS Direct Connect costs $1,500–$2,250/mo; Azure ExpressRoute $5,000–$8,000/mo — Bit Refinery: included free.
- [Intelligent Hosting (MetaCato)](https://bitrefinery.com/services/intelligent-hosting): Federated infrastructure intelligence platform using CatoInsights. Hub-and-spoke model centralizes intelligence while data stays where it belongs. Behavioral security, verified ROI, evidence-based architecture recommendations. Zero friction — connects to existing systems.
- [Managed IT Solutions (LoDo IT)](https://bitrefinery.com/services/managed-it-solutions): End-to-end managed IT services — 24/7 help desk, cybersecurity, cloud solutions, active monitoring on Bit Refinery bare metal. Supports 10 to 300+ users. AI-powered business automation with 40% average productivity gains.
- [Free Infrastructure Tools](https://bitrefinery.com/tools): 24 free tools including speed test, DNS lookup, cloud cost calculator, egress fee calculator, server sizing tool, GPU cost calculator, RAID calculator, SSL checker, and more.
