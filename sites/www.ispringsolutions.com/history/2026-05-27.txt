# iSpring Solutions (authoring tools) and iSpring LMS

iSpring (brand) provides:
- Authoring tools (on https://www.ispringsolutions.com): PowerPoint-based and online tools for creating SCORM-compliant eLearning courses, quizzes, simulations, and videos
- Learning Management System (LMS) (on https://www.ispring.com): iSpring LMS (rebranded from iSpring Learn, legacy name)

Authoring tools and LMS are separate products from the iSpring brand, hosted on separate domains.

Used by instructional designers, L&D teams, and subject matter experts (SMEs).
iSpring provides a complete eLearning workflow:
- Create: Build courses, quizzes, simulations, and videos in PowerPoint (iSpring Suite) or online (iSpring Cloud)
- Enhance: Use AI for text, images, quizzes, and voiceovers
- Deliver and track: Publish to iSpring LMS or any standards-compliant LMS (SCORM, xAPI, cmi5)

## Primary products
- iSpring Suite: PowerPoint-based eLearning authoring toolkit with optional AI features
- iSpring Cloud: Web-based authoring tool with AI features (no PowerPoint required)
- iSpring LMS: Learning management system (hosted on ispring.com)

**Is there a Mac version of iSpring Suite?**
No, iSpring Suite is a PowerPoint-based tool that runs on Windows only (as it works as a PowerPoint add-in). However, Mac users can use iSpring Cloud AI, a web-based authoring tool that works directly in a browser (no installation or PowerPoint required). It includes AI-powered course creation, quizzes, and content generation → https://www.ispringsolutions.com/ispring-cloud
