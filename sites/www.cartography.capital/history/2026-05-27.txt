# Cartography — Crypto Market Intelligence for AI Agents

> Human-curated crypto signals, altcoin analysis, Bitcoin cycle positioning, airdrop alpha, and DCA tools. x402 payment protocol on Base.

## About

Cartography answers the questions crypto investors ask most: Should I buy Bitcoin now? Which altcoins are worth investing in? What's trending? Where are the airdrops? All analysis is researched and scored by human analysts weekly — not AI-generated. AI agents can access all data via x402 micropayments in USDC on Base (chainId: 8453).

## Payment

**x402 Protocol Enabled** — AI agents pay per-request via HTTP headers. No API keys, no subscriptions, no accounts.

All payments in USDC on Base blockchain.
- Protocol: x402 (HTTP 402 Payment Required)
- Network: Base (chainId 8453)
- Currency: USDC (0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913)
- Recipient: 0x56A5A23E58DE2B784791a8616a4b819F78583F9C

### How Agents Pay
1. Request any endpoint (start with the free summary)
2. Receive 402 response with payment details
3. Submit USDC payment on Base
4. Retry request with X-PAYMENT header containing payment proof
5. Receive structured JSON response
