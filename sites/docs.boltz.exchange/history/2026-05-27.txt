# Boltz Documentation

> This documentation is written for anyone looking to learn about Boltz, how to integrate Boltz Swaps via the public API or use Boltz with one of the clients

## Products

- [👨‍💻 Boltz API](https://api.docs.boltz.exchange/llms.txt): If you are looking to integrate Boltz Swaps into your application, then check out our Boltz API docs
- [🤖 Boltz Client](https://client.docs.boltz.exchange/llms.txt): If you are looking to use Boltz to conveniently manage liquidity of your CLN or LND nodes, then check out our Boltz Client docs
- [💻 Boltz Web App](https://web.docs.boltz.exchange/llms.txt): If you are a self-sovereign individual looking to run Boltz Web App yourself from source instead of using boltz.exchange, then check out our Boltz Web App docs
- [💚 Boltz BTCPay Plugin](https://btcpay.docs.boltz.exchange/llms.txt): If you are a merchant looking to accept Lightning payments in an incredibly simple, yet non-custodial way then check out our Boltz BTCPay Plugin docs
