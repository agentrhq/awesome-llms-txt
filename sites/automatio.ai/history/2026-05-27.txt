# Automatio

> Automatio is a general-purpose, no-code AI automation and data extraction platform
> that lets anyone automate any website using plain English commands. The platform
> intelligently routes requests to 11 specialized AI agents with 56+ tools covering
> web scraping, email, spreadsheets, calendars, browser automation, voting, crypto
> data, image generation, and code generation. No technical skills required.
> Available at https://automatio.ai

## Getting Started

- [Home](https://automatio.ai/): Platform overview with agent showcase, use cases, and pricing
- [Register](https://automatio.ai/register): Create a free account to start automating
- [Login](https://automatio.ai/login): Sign into your Automatio account
- [Chat Interface](https://automatio.ai/chat): Main automation interface — describe any task in plain English and the right agent is automatically selected

## AI Agents & Capabilities

Automatio uses LLM-based routing to automatically select the best agent for each request.
All 11 agents are available within the same chat interface at https://automatio.ai/chat

- **Web & SEO Specialist** — READ-ONLY web access: search the web, scrape websites, analyze SEO metrics, extract data from YouTube, Twitter/X, and Reddit. Cannot fill forms or click buttons (use Browser Automation for that).
- **Gmail Specialist** — Send emails, read inbox, search messages, manage labels, organize threads, handle attachments, create drafts. Requires Gmail OAuth connection.
- **Google Sheets Specialist** — Read, write, create, and modify Google Sheets. Import/export data, build pipelines between sheets.
- **Google Calendar Specialist** — Create, update, and read calendar events. Manage schedules, set reminders, handle invites.
- **Browser Automation Specialist** — INTERACTIVE browser control: fill forms, click buttons, type text, submit data, login to any website, take screenshots, navigate multi-step flows.
- **Ghost CMS Specialist** — Manage Ghost CMS content: create and publish posts, manage pages, update content programmatically.
- **Voting Specialist** — Automate vote submissions on major polling platforms by URL.
- **Crypto Agent** — Real-time cryptocurrency prices, market cap data, trending tokens, meme coin discovery, token research.
- **Image Generation Specialist** — Generate images from text prompts or edit/modify attached images using AI.
- **Coding Specialist** — Write, debug, and explain code in any language: HTML, CSS, JavaScript, TypeScript, Python, SQL, bash, and more.
- **General Assistant** — Conversational AI for general questions, explanations, and tasks that don't require specialized tools.
