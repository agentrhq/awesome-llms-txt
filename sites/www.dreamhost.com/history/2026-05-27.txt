## Supported Technologies

### Container & Orchestration
- **Docker**: Not supported on Shared/Managed VPS Hosting. Manual installation possible on Dedicated Servers with admin user or DreamCompute
- **Kubernetes**: Available only on DreamCompute cloud platform
- **Note**: Container technologies require root/sudo access, limiting them to Dedicated Servers with admin user or DreamCompute

### Databases
- **MySQL 8**: Fully supported on all hosting plans
- **PostgreSQL**: Not supported on Shared/VPS. Manual installation on Dedicated Servers or DreamCompute only
- **MongoDB/NoSQL**: Not supported on managed hosting. Manual installation on Dedicated Servers or DreamCompute only
- **SQLite3**: Supported on all servers
- **Redis**: Available on DreamPress Pro plans, manual installation possible on Dedicated/DreamCompute
- **Backup retention**: 5-day retention for databases (vs 2-week for files)

### Runtime Environments
- **PHP**: 8.x current standard, PHP 7.x Extended Support available
- **Python**: 3.x versions including 3.10.1, 3.9.2, 3.6.2 (virtual environment required)
- **Ruby**: Full support with Ruby on Rails (VPS/Dedicated servers required)
- **Node.js**: Supported on VPS and Dedicated Servers (not Shared hosting)
- **Java JRE**: Supported on all servers
- **Go**: Manual installation under username (Dedicated/DreamCompute if sudo required)
- **Erlang/Elixir/Phoenix**: Manual installation on VPS/Dedicated (sudo required for full setup)

### Development Tools
- **Git**: Installed on all servers
- **cURL**: Installed on all servers  
- **Composer/npm/yarn**: Supported for local installation
- **WebSockets**: Supported on VPS/Dedicated using non-privileged ports (no public proxy server support)
- **Apache mods**: mod_rewrite (YES), mod_security (YES), mod_perl (NO), mod_proxy (VPS/Dedicated only)

### Platform Limitations by Hosting Type
- **Shared Hosting**: No root/sudo, no Docker, no Node.js, no PostgreSQL, memory-intensive apps restricted (including WooCommerce)
- **VPS**: No root/sudo, persistent processes allowed, Node.js supported, custom Apache/Nginx config
- **Dedicated with Admin**: Full root/sudo access, Docker/Kubernetes installable, all databases supported
- **DreamCompute**: Full control, Docker/Kubernetes supported, any Linux software installable
- **WordPress Multisite**: Subdomain setup requires VPS/Dedicated (wildcard DNS needed), subdirectory setup works on Shared

### Not Supported
- **.NET/ASP**: No Microsoft technologies on Linux servers
- **Tomcat**: Requires Dedicated/DreamCompute for installation
- **ColdFusion**: Requires Dedicated/DreamCompute
- **Plesk/cPanel**: DreamHost uses custom control panel (Plesk only on DreamCompute)
- **FTPS**: Not supported (use SFTP instead)
- **Passenger**: Not supported on DreamHost servers
