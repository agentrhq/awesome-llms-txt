## Main Features
- [Features Overview](https://www.deployhq.com/features): Core features and capabilities of DeployHQ
- [Automatic Deployment](https://www.deployhq.com/features/automatic-deployments): Set up automated deployments with DeployHQ
- [Build Pipelines](https://www.deployhq.com/features/build-pipelines): Configure build processes with DeployHQ
- [Deploy Behind Firewalls](https://www.deployhq.com/features/deploy-behind-firewalls): Deploy to protected environments with DeployHQ
- [DeployHQ AI](https://www.deployhq.com/features/ai): AI-powered deployment intelligence, summaries, and risk assessment
- [DeployHQ MCP Server](https://www.deployhq.com/features/deployhq-mcp): Model Context Protocol server for Claude, Cursor, and Windsurf
- [Deployment Checks](https://www.deployhq.com/features/deployment-checks): Gate every deploy with pre-build and post-deploy validations — SSH commands, HTTP probes, and vulnerability scans (Trivy, Snyk)
- [Deployment Targets](https://www.deployhq.com/features/deployment-targets): Configure deployment destinations with DeployHQ
- [Deployment Templates](https://www.deployhq.com/features/deployment-templates): Create reusable deployment configs with DeployHQ
- [Deployment Zones](https://www.deployhq.com/features/deployment-zones): Deploy from multiple global regions
- [Docker Builds](https://www.deployhq.com/features/docker-builds): Build projects using Docker containers
- [One-Click Rollback](https://www.deployhq.com/features/one-click-rollback): Instantly revert to a previous deployment
- [SSH Deployment](https://www.deployhq.com/features/ssh-deployment): Deploy to any SSH-accessible server (VPS, dedicated, or shared hosting) over SSH or SFTP with key-based authentication
- [Turbo Deployments](https://www.deployhq.com/features/turbo-deployments): Accelerated file transfers for faster deployments
- [Zero Downtime Deployments](https://www.deployhq.com/features/zero-downtime-deployments): Deploy without service interruption with DeployHQ
- [Team & Permissions](https://www.deployhq.com/features/team-permissions): Role-based access control with three permission layers (account, project, server) and Teams on Business and Enterprise plans
- [Static Site Deployment](https://www.deployhq.com/features/static-hosting): Deploy static sites to Amazon S3, Cloudflare R2, DigitalOcean Spaces, and S3-compatible storage
- [API](https://www.deployhq.com/features/api): RESTful API for programmatic access to DeployHQ
- [Integrations](https://www.deployhq.com/features/integrations): Connect DeployHQ with third-party services
