# EduBase

> EduBase is a comprehensive e-learning platform that unifies online exams, videos, SCORM content, and learning materials in a single system. It features advanced capabilities including parametric question generation with 15+ question types, real-time cheating detection, AI-powered content creation, enterprise-grade security (LDAP/AD/SSO/2FA), and comprehensive APIs. Available as cloud-hosted or on-premise with custom branding, it serves educational institutions, businesses, and individual educators with a modular architecture designed for scalability and integration.

## Core Products
- [EduBase Platform](https://www.edubase.net): The unified learning management system combining Quiz, content management, and organization tools
- [EduBase Quiz](https://www.edubasequiz.com): Specialized advanced quizzing system with parametrization and anti-cheating capabilities
- [EduBase AI Assistant](https://www.edubase.net/assistant): AI-powered tool for question generation and translation from text, presentations, or images

## Advanced Question Generation
- **20+ Question Types**: CHOICE, MULTIPLE-CHOICE, TRUE/FALSE, NUMERICAL, EXPRESSION, ORDER, GROUPING, PAIRING, MATRIX, MATRIX:EXPRESSION, SET, SET:TEXT, DATE/TIME, TEXT, FREE-TEXT, FILE, HOTSPOT, READING, GENERIC
- **Parametric Questions**: Create infinite variations with the same difficulty level using:
  - Parameters Types: FIX, INTEGER, FLOAT, FORMULA, LIST, PERMUTATION, FORMAT
  - Parameter Constraints: Mathematical expressions ensuring parameter validity
  - Parameter Synchronization: Coordinated selection from LIST parameters
- **Math & Science Support**:
  - LaTeX integration for mathematical formulas and equations
  - Expression evaluation with multiple validation methods (RANDOM, EXPLICIT, COMPARE)
  - Support for matrices, vectors, and mathematical functions
- **Scoring Options**:
  - Partial credit with multiple methods (PROPORTIONAL, LINEAR_SUBSTRACTED, CUSTOM, NONE)
  - Custom point distribution with SUBPOINTS
  - Penalty scoring for incorrect answers (PER_ANSWER or PER_QUESTION)
  - Assistance penalties for hints, solutions, and videos
- **Answer Processing**:
  - Customizable answer tolerance (ABSOLUTE, RELATIVE, QUOTIENT)
  - Support for answer ranges and intervals
  - Configurable precision for numerical answers

## Solutions

### Industrial Safety Compliance Made Simple
> Automated digital contractor induction: safety training and compliance management with instant competency verification, tamper-proof certificates, and audit-ready records across all sites.
#### Overview
EduBase is a cloud-based contractor induction and compliance management platform used by 210+ companies, with 440,000+ successful exams delivered since 2016. Contractors access training by scanning a QR code — no app, account, or password required. All events are logged and timestamped for audit readiness.
#### Key Problems Solved
- **Audit readiness**: centralized, standardized certificate storage eliminates last-minute scrambling before inspections.
- **Project delays**: eliminates manual verification bottlenecks and long contractor wait times.
- **Administrative burden**: automates expiry tracking, renewal reminders, and duplicate data across systems.
#### Core Features
- QR code-based access — scan, test, and get certified in minutes, no installation needed
- Complete contractor histories with timestamped audit logs
- Multi-site dashboard with standardized training and site-specific variations
- Automated renewal deadline alerts and real-time qualification status
- Integrations: access control systems, HRIS platforms, LTI 1.3, REST API / Webhooks
- Content support: videos, documents, SCORM, interactive formats
- AI features: multilingual translation, adaptive chat-based assessments, content analysis
#### Deployment Options
- **Cloud**: GDPR-compliant, EU data residency, zero infrastructure management, automatic updates
- **On-premise**: air-gapped / classified environments, full offline operation
#### Security & Compliance
- GDPR compliant, EU data residency, encryption in transit and at rest
- Enterprise identity: LDAP, Active Directory, SSO, optional 2FA
- Role-based access control with detailed audit logs
#### Pricing & Onboarding
- Pricing based on Monthly Active Users (MAU) or managed user count; volume discounts available; no setup fee or long-term contracts for cloud
- Implementation typically under one week; initial onboarding and training included
- 14-day free trial available with a dedicated EduBase instance
#### Developer Resources
- 150+ API endpoints documented at [developer.edubase.net](https://developer.edubase.net/)
- Supports SSO, AD/LDAP, LTI 1.3, REST API and Webhooks
