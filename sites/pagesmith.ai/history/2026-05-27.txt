# Pagesmith

> Pagesmith is an AI website builder that generates production-ready Astro sites from a text description. Sites are built for SEO from day one — optimized for Google, AI search engines like ChatGPT and Perplexity, and deployed to Cloudflare's edge network. Users own the code and can export it anytime.

Pagesmith generates static Astro sites with server-side rendering, semantic HTML, schema markup (JSON-LD), Open Graph tags, sitemaps, and robots.txt out of the box. Sites score 90+ on Lighthouse. Every page is designed to be indexed by search engines and cited by AI assistants.

Key capabilities:
- AI site generation from natural language descriptions
- Built-in SEO optimization (meta tags, schema markup, sitemaps, semantic HTML)
- AI search optimization (GEO/AEO) for ChatGPT, Perplexity, Google AI Overviews
- CMS with blog support and structured data
- Custom domain connection with automatic SSL
- Code export to GitHub (full Astro source code)
- Site import — paste a URL and Pagesmith recreates it
- Form handling with submissions dashboard
- Edge hosting on Cloudflare Workers (global CDN)
