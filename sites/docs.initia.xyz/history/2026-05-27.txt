- [Compute Route](https://docs.initia.xyz/api-reference/routing/compute-route.md): This is the first step when bridging tokens between chains. Call this endpoint with your source and destination assets to find the optimal transfer route. The response includes the expected output amount, USD value estimates, and an ordered list of operations.
- [Generate Transaction](https://docs.initia.xyz/api-reference/routing/generate-transaction.md): This is the second step when bridging tokens. After computing a route with Compute Route (`POST /v2/fungible/route`), pass the route details and your addresses to this endpoint. It returns signable transaction messages that you can broadcast to the source chain.
- [Get OP hook](https://docs.initia.xyz/api-reference/routing/get-op-hook.md): Generates OP hook data required for certain cross-chain routes that involve Optimistic bridge withdrawals with ERC-20 token conversion. When the Compute Route response includes `required_op_hook: true`, you must call this endpoint to obtain the hook payload before generating transaction messages.
- [Get Transaction Status](https://docs.initia.xyz/api-reference/tracking-transactions/get-transaction-status.md): Polls the current status of a transaction that was previously registered with Track Transaction (`POST /v2/tx/track`). Returns the overall status and the state of each hop in the transfer sequence.
- [Track Transaction](https://docs.initia.xyz/api-reference/tracking-transactions/track-transaction.md): Registers a transaction for cross-chain tracking. Call this once with the transaction hash and chain ID immediately after broadcasting. This is required before you can poll the transaction status.
- [Get transaction by hash](https://docs.initia.xyz/api-reference/transactions/get-transaction-by-hash.md): Returns a specific transaction by its hash.
- [Get transactions](https://docs.initia.xyz/api-reference/transactions/get-transactions.md): Returns a paginated list of transactions. You can optionally filter by message type.
- [Get transactions by account](https://docs.initia.xyz/api-reference/transactions/get-transactions-by-account.md): Returns transactions associated with a specific account address.
- [Get transactions by height](https://docs.initia.xyz/api-reference/transactions/get-transactions-by-height.md): Returns all transactions at a given block height.
