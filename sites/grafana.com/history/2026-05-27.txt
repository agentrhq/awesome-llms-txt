# Grafana documentation

> Grafana Labs builds open and composable observability software, including Grafana, Grafana Cloud, Loki, Mimir, Tempo, Pyroscope, Alloy, Beyla, Faro, and k6.

This file is a curated documentation index for AI agents and other automated consumers.

Use this file for discovery. The links below point directly to machine-readable Markdown pages on `https://grafana.com`.

For a larger exhaustive index, use [llms-full.txt](/llms-full.txt).

All Grafana documentation pages support a machine-readable Markdown form:
- Append `.md` to the canonical docs URL.
- Or request the canonical docs URL with `Accept: text/markdown`.

Examples:
- HTML: `https://grafana.com/docs/grafana-cloud/developer-resources/api-reference/http-api/`
- Markdown: `https://grafana.com/docs/grafana-cloud/developer-resources/api-reference/http-api.md`
