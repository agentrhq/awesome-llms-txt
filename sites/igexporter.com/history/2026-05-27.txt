# IG Exporter

> A browser extension for exporting Instagram followers and following data to CSV/Excel. Available for Chrome and Edge with 400,000+ users.

IG Exporter is a browser extension that allows users to export Instagram followers and following lists to CSV or Excel formats. The extension works in Google Chrome, Microsoft Edge, and other Chromium-based browsers.

Key features include exporting up to 50,000 followers, detailed profile data extraction (bio, email, phone, website), proper CSV/Excel formatting, and rate limit management with cooldown mode. Free tier supports up to 550 followers, with Pro tier offering unlimited exports.

## Features

- [Export followers/following](https://igexporter.com/instagram-follower-export-tool/): Export up to 50,000 followers and following lists
- [Detailed Mode](https://igexporter.com/instagram-audience-analysis-tool/): Extract rich profile data including bio, email, phone, website for outreach and analysis
- [CSV/Excel Export](https://igexporter.com/export-instagram-data-csv/): Proper formatting compatible with modern office applications
