# Gaslighting Check

> Gaslighting Check is an AI-powered tool that analyzes text, audio, and conversations to detect manipulation tactics like gaslighting, blame shifting, and emotional abuse. Features AI coach for personalized guidance and building a supportive community for abuse survivors.

## Features

- [Gaslighting Check](https://www.gaslightingcheck.com/dashboard/check): Analyze text, audio, and files for gaslighting patterns
- [AI Coach](https://www.gaslightingcheck.com/dashboard/ai-coach): Interactive assistant for processing abuse experiences and discussing analysis results
- [Blog](https://www.gaslightingcheck.com/blog): Educational content about gaslighting and recovery
