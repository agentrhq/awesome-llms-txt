# Abstract API

## Docs

- [🚀 Welcome to Abstract API](https://docs.abstractapi.com/api.md): Explore all APIs, guides, and resources to integrate Abstract into your applications.
- [Avatars API](https://docs.abstractapi.com/api/avatars.md): Abstract's Avatars API is a fast, lightweight, modern, and RESTful JSON API for creating highly customizable avatar images with a person's name or initials to improve your user experience.
- [Company Enrichment API](https://docs.abstractapi.com/api/company-enrichment.md): With Abstract's Company Enrichment API, you can easily get detailed information about any company domain.
- [Email Reputation API](https://docs.abstractapi.com/api/email-reputation.md): Improve your delivery rates, clean your email lists, and block fraudulent users with Abstract’s industry-leading Email Reputation API.
- [Exchange Rates API](https://docs.abstractapi.com/api/exchange-rates.md): Abstract's Exchange Rates API is a fast, lightweight, modern, and RESTful JSON API for looking up the latest exchange rates for 80+ currencies, getting historical exchange rates, and converting an arbitrary amount from one currency to another.
- [Public Holidays API](https://docs.abstractapi.com/api/holidays.md): Abstract's Public Holidays API is a fast, lightweight, modern, and RESTful JSON API that allows you to get the public, local, religious, and other holidays of a particular country.
- [IBAN Validation API](https://docs.abstractapi.com/api/iban-validation.md): Abstract's IBAN Validation API is a fast, lightweight, modern, and RESTful JSON API for determining the validity and other details of the International Bank Account Number (IBAN).
- [Image Processing API](https://docs.abstractapi.com/api/images.md): Abstract's Image Processing and Optimization API is a simple yet powerful REST API that is used to convert, compress, or optimization an image that is provided via a URL or via POST to the API itself.
- [IP Intelligence API](https://docs.abstractapi.com/api/ip-intelligence.md): Abstract's IP Intelligence API is a fast, reliable, and RESTful JSON API that provides advanced security and geographical insights about IP addresses worldwide. It identifies VPNs, proxies, TOR usage, abuse potential, hosting services, relays, and mobile IPs.
- [Phone Intelligence API](https://docs.abstractapi.com/api/phone-intelligence.md): Abstract's Phone Intelligence API is a fast, lightweight, modern, and RESTful JSON API for retrieving detailed intelligence about phone numbers, including carrier, location, messaging, registration, risk, and breach data.
- [Web Scraping API](https://docs.abstractapi.com/api/scrape.md): Abstract’s Web Scraping API is a simple yet powerful REST API used to extract data from a given URL. To make a request, you simply include the target URL and your API key, and Abstract’s API will return the data from that site.
- [Website Screenshot API](https://docs.abstractapi.com/api/screenshot.md): Abstract's Website Screenshot API is a simple yet powerful REST API that allows you to request an image of a given URL. It accepts but does not require various customizations on the image capture and rendering, such as setting a capture delay, injecting custom CSS, and specifying the capture's view…
- [Timezone API](https://docs.abstractapi.com/api/timezones.md): The Time, Date, and Timezone API is the easiest way to find, convert, and manage time and timezone data across the world.
- [VAT Validation API](https://docs.abstractapi.com/api/vat-validation.md): Abstract's VAT Validation API is a fast, lightweight, modern, and RESTful JSON API that allows you to stay compliant for all your domestic and cross-border sales.
