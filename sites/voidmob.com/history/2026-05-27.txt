# VoidMob

> VoidMob is a privacy-focused mobile services platform that provides real 4G/5G mobile proxies, non-VoIP SMS verification (real SIM cards), and global eSIM data plans through a unified dashboard. All services run on carrier infrastructure with instant activation, no KYC requirements, and cryptocurrency payment support. VoidMob also offers an open-source MCP server (`@voidmob/mcp`) that gives AI agents programmatic access to all services, and a suite of free privacy tools for checking IP types, browser fingerprints, WebRTC leaks, and platform detection signals.

VoidMob serves individuals and businesses who need digital privacy, security researchers, e-commerce sellers managing multiple marketplace accounts, social media managers, AI agent developers, and international travelers. The platform differentiates itself by combining three typically separate services (proxies, SMS, eSIM) into one account with one balance, and by using real mobile carrier infrastructure rather than datacenter or VoIP-based alternatives.

## Core Services

- [Mobile Proxies](https://voidmob.com/proxies): Real 4G/5G carrier IPs from mobile devices. Pay-per-GB or dedicated monthly proxies. Unlimited bandwidth, sticky sessions, global coverage. HTTP(S) and SOCKS5 protocols.
- [SMS Verification](https://voidmob.com/sms): Non-VoIP US phone numbers from real carrier-issued SIM cards for SMS verification. 95%+ success rate on platforms that block virtual numbers. Instant OTP delivery, single-use or long-term rental options.
- [Global eSIM](https://voidmob.com/esim): Instant eSIM activation in 200+ destinations. No physical cards, no shipping. Data plans for travelers and privacy-conscious users who need connectivity without tying identity to a local carrier.

## MCP Server (AI Agent Integration)

- [MCP Integration Page](https://voidmob.com/mcp): Open-source Model Context Protocol server with 18 tools across 5 categories (mobile proxies, SMS verification, eSIM plans, wallet, orders). Works with Claude Code, Claude Desktop, Cursor, and any MCP-compatible client. MIT licensed. Install: `npx -y @voidmob/mcp`. Sandbox mode available with $50 test balance, no API key needed.
- [GitHub Repository](https://github.com/voidmobcom/voidmob-mcp): Source code, documentation, and issues for the VoidMob MCP server.
- [npm Package](https://www.npmjs.com/package/@voidmob/mcp): `@voidmob/mcp` on the npm registry.

## Free Privacy Tools

- [IP Type Checker](https://voidmob.com/tools/ip-checker): Check if your IP is mobile, residential, datacenter, or VPN. Shows location, carrier, and ASN.
- [Browser Fingerprint Test](https://voidmob.com/tools/fingerprint-test): See how unique and trackable your browser is. Checks canvas, WebGL, fonts, audio context.
- [WebRTC Leak Test](https://voidmob.com/tools/webrtc-leak-test): Check if WebRTC is leaking your real IP address through VPN or proxy.
- [Location Consistency Test](https://voidmob.com/tools/location-consistency-test): Check if your IP location matches your browser timezone and language settings.
- [Platform Detection Simulator](https://voidmob.com/tools/platform-detection): See what Instagram, TikTok, and other platforms can detect about your browser, IP, and device.
- [Platform Trust Score](https://voidmob.com/tools/platform-trust-score): Combined trust score from IP, fingerprint, WebRTC, and location signals. Shows if platforms will flag your setup.
- [Travel Data Calculator](https://voidmob.com/tools/travel-data-calculator): Estimate how much mobile data you need based on destination, duration, and usage habits.
- [eSIM vs Roaming Calculator](https://voidmob.com/tools/esim-roaming-calculator): Compare eSIM costs against AT&T, Verizon, T-Mobile international roaming fees.
